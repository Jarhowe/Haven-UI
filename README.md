## 安装
**克隆项目**        
`git clone https://github.com/Jarhowe/Haven-UI.git`

**进入项目目录**        
`cd Haven-UI`

**使用npm或者yarn安装依赖**        
`npm install`

**启动文档预览**        
`npm run docs:dev`

浏览器访问 http://localhost:8080
